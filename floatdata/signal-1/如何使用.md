# 📖 Signal-1 数据使用指南

## 🎯 在原系统中使用

### 方式一：使用批处理脚本（推荐）

```bash
# 进入系统目录
cd e:\Code\CW_Cloud\floatdata\floatdata-streaming

# 双击运行
view-signal-analysis.bat

# 选择 [1] Signal-1 数据
```

### 方式二：使用命令行

```bash
# 进入系统目录
cd e:\Code\CW_Cloud\floatdata\floatdata-streaming

# 运行可视化命令
python tdms-reader.py --visualize -s 100000

# 或指定具体文件
python tdms-reader.py --visualize --file "..\signal-1\ae_sim_2s.tdms" -s 100000
```

### 方式三：在Python代码中调用

```python
from tdms_reader import visualize_tdms_signals

# 可视化信号
visualize_tdms_signals(
    file_path="e:/Code/CW_Cloud/floatdata/signal-1/ae_sim_2s.tdms",
    sample_rate=100000
)
```

## 📊 输出结果

运行后会生成：
- **图片文件**: `ae_sim_2s_filter_analysis.png`
  - 4个子图展示完整的信号处理流程
  - 自动保存在 signal-1 文件夹中
  - 高清图片（300 DPI）

- **控制台输出**:
  - 通道信息
  - 滤波器参数
  - 性能指标（MSE改善百分比）

## 🔧 命令行参数

```bash
python tdms-reader.py --help

可选参数:
  -v, --visualize        启用可视化模式
  -f, --file PATH        指定TDMS文件路径
  -s, --sample-rate HZ   采样率（Hz），默认100000
```

## 📋 示例

### 示例1：默认可视化

```bash
python tdms-reader.py -v
```

自动查找并分析 signal-1 数据。

### 示例2：指定文件和采样率

```bash
python tdms-reader.py -v -f "..\signal-1\ae_sim_2s.tdms" -s 100000
```

### 示例3：正常传输模式（不可视化）

```bash
python tdms-reader.py
```

将数据传输到Netty服务器（原有功能保持不变）。
