## 🎯 核心问题：异常结果在哪里？

### 简短回答

**异常结果在3个地方**：

1. **生成位置**: `AnomalyDetector.detect()` 方法（第50-56行）
2. **处理位置**: `StreamProcessor.processStream()` 方法（第120-167行）
3. **输出位置**:

## 📍 详细位置说明

### 1. 异常结果生成（核心代码）

**文件**: `src/main/java/com/floatdata/processor/AnomalyDetector.java`

```java
public AnomalyResult detect(SignalData signalData) {
    float[] samples = signalData.getSamples();
    
    // 步骤1: Butterworth滤波
    float[] filtered = filter.applyButterworthFilter(samples);
    
    // 步骤2: 计算能量
    double energy = filter.calculateEnergy(filtered);
    double normalizedEnergy = normalizeEnergy(energy);
    
    // 步骤3: FFT频率分析
    double[] frequencyMagnitude = filter.calculateFrequencyFeatures(filtered);
    double frequencyScore = filter.calculateFrequencyScore(frequencyMagnitude);
    
    // ⭐⭐⭐ 步骤4: 创建异常结果对象（这就是异常结果！）
    AnomalyResult result = new AnomalyResult(
        signalData.getTimestamp(),    // 时间戳
        signalData.getSensorId(),     // 传感器ID
        signalData.getLocation(),     // 位置
        normalizedEnergy,             // 能量水平 (0-1)
        frequencyScore                // 频率得分 (0-1)
    );
    
    return result;  // 返回异常结果
}
```

### 2. 异常结果数据结构

**文件**: `src/main/java/com/floatdata/utils/AnomalyResult.java`

```java
public class AnomalyResult {
    private long timestamp;           // 时间戳
    private int sensorId;            // 传感器ID  
    private String location;         // 位置
    private double energyLevel;      // 能量水平 (0-1)
    private double frequencyScore;   // 频率得分 (0-1)
    private double anomalyScore;     // 综合异常分数 (0-1) ⭐
    private boolean isAnomaly;       // 是否异常 ⭐⭐⭐
    private String anomalyType;      // 异常类型 ⭐
    private long processingTime;     // 处理时间(ms)
}
```

**异常判定逻辑**（在构造函数中）：
```java
// 计算综合异常分数
this.anomalyScore = (energyLevel + frequencyScore) / 2.0;

// 判定是否异常（阈值：0.75）
this.isAnomaly = this.anomalyScore > 0.75;

// 分类异常类型
private String determineAnomalyType() {
    if (!isAnomaly) return "NORMAL";              // 正常
    if (energyLevel > 0.8) return "HIGH_ENERGY";  // 高能量异常
    if (frequencyScore > 0.8) return "FREQUENCY_ANOMALY"; // 频率异常
    return "MIXED_ANOMALY";                       // 混合异常
}
```

### 3. 异常结果处理流程

**文件**: `src/main/java/com/floatdata/spark/StreamProcessor.java`

```java
// ⭐ 步骤1: 在Spark中并行执行异常检测
JavaDStream<AnomalyResult> resultStream = segmentedStream
    .flatMap(signalList -> {
        // 每个Executor进程独立运行
        AnomalyDetector detector = new AnomalyDetector();
        List<AnomalyResult> results = new ArrayList<>();
        
        for (SignalData signal : signalList) {
            AnomalyResult result = detector.detect(signal); // 生成异常结果
            results.add(result);
        }
        
        return results.iterator();
    });

// ⭐ 步骤2: 输出异常结果到多个渠道
resultStream.foreachRDD(rdd -> {
    rdd.foreach(result -> {
        // 输出1: 发送到Kafka
        sendResultToKafka(result);
        
        // 输出2: 记录异常日志
        if (result.isAnomaly()) {
            logger.warn("检测到异常: {}", result);
        }
    });
    
    // 输出3: 批次统计
    long count = rdd.count();
    long anomalyCount = rdd.filter(AnomalyResult::isAnomaly).count();
    logger.info("批次处理: 总数={}, 异常={}, 异常率={:.2f}%",
            count, anomalyCount, (double) anomalyCount / count * 100);
});

// ⭐ 步骤3: 周期统计（每10秒）
resultStream.window(Durations.seconds(10))
    .foreachRDD(rdd -> {
        Map<String, Long> anomalyTypeCount = rdd
                .filter(AnomalyResult::isAnomaly)
                .mapToPair(r -> new Tuple2<>(r.getAnomalyType(), 1L))
                .reduceByKey((a, b) -> a + b)
                .collectAsMap();
        
        logger.info("异常类型统计: {}", anomalyTypeCount);
    });
```

### 方法3: 编程方式消费Kafka

创建消费者程序读取异常结果：

```java
// 文件: src/main/java/com/floatdata/client/AnomalyResultConsumer.java
public class AnomalyResultConsumer {
    public static void main(String[] args) {
        Properties props = new Properties();
        props.put("bootstrap.servers", "localhost:9092");
        props.put("group.id", "anomaly-consumer");
        props.put("key.deserializer", StringDeserializer.class);
        props.put("value.deserializer", StringDeserializer.class);
        
        KafkaConsumer<String, String> consumer = new KafkaConsumer<>(props);
        consumer.subscribe(Collections.singletonList("anomaly-detection-result"));
        
        while (true) {
            ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));
            for (ConsumerRecord<String, String> record : records) {
                // 解析异常结果
                AnomalyResult result = AnomalyResult.fromJson(record.value());
                
                // 处理异常
                if (result.isAnomaly()) {
                    System.out.println("⚠️ 异常告警: " + result);
                    // 可以添加：发送邮件、短信、推送通知等
                }
            }
        }
    }
}
```

## 🎯 总结

### 异常结果的完整路径

```
1. 数据采集 (样机/模拟器)
   └─ 发送到 Netty:9090

2. Netty接收
   └─ 解析并发送到 Kafka Topic: acoustic-emission-signal

3. Spark Streaming消费
   └─ 从Kafka读取数据

4. 数据分段
   └─ 窗口分批 → 多个Partition

5. 并行处理（多进程）
   ├─ Executor 1 → AnomalyDetector.detect() → 生成AnomalyResult
   ├─ Executor 2 → AnomalyDetector.detect() → 生成AnomalyResult
   └─ Executor 3 → AnomalyDetector.detect() → 生成AnomalyResult

6. 异常结果输出 ⭐⭐⭐
   ├─ Kafka Topic: anomaly-detection-result
   ├─ 日志文件: logs/spark-streaming.log  
   ├─ 批次统计: 每2秒输出一次
   └─ 周期统计: 每10秒输出一次
```
