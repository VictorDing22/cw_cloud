# .gitignore 优化说明

## 📋 优化目标
保留项目必需的大文件，忽略可重新生成或下载的文件，减少仓库大小。

---

## ✅ 已优化的忽略规则

### 1. **Kafka & Zookeeper 相关**
```gitignore
# 压缩包（可重新下载）
kafka_*.zip
kafka*.tar.gz
nacos-server-*.zip

# 运行时数据（自动生成）
**/data/kafka-logs/
**/data/zookeeper/

# 日志文件
kafka/logs/
**/kafka/logs/
```

### 2. **编译产物**
```gitignore
# Java编译产物
**/target/
*.jar  # 在target目录下的
*.war
*.ear
```

### 3. **下载的压缩包**
```gitignore
# 所有可重新下载的压缩包
*.zip
*.tar.gz
*.tgz
```

## 📝 注意事项

### 如果需要提交特定的zip文件
在 .gitignore 的"保留重要文件"部分添加：
```gitignore
!特定文件名.zip
```

### 如果需要忽略特定的TDMS文件
单独添加规则：
```gitignore
floatdata/data/不需要的文件.tdms
```
