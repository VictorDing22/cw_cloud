## 🎯 **Phase 1: 急迫优化 (1-2天)**

### **✅ 立即执行清单**

#### **步骤1: 数据库表创建**
```bash
# 1. 连接MySQL数据库
mysql -u root -p

# 2. 执行表创建脚本
source create-filter-tables.sql

# 3. 验证表创建
SHOW TABLES LIKE '%filter%';
SHOW TABLES LIKE '%anomaly%';
```

#### **步骤2: 安装WebSocket Bridge依赖**
```bash
# 安装mysql2依赖
npm install mysql2

# 重启WebSocket Bridge
taskkill /F /IM node.exe  # 停止现有进程
node websocket-bridge.js  # 启动新版本
```

#### **步骤3: backend.jar Kafka配置**
```bash
# 1. 将backend-kafka-config.yml内容添加到backend.jar配置
# 2. 重启backend.jar并指定配置
java --add-opens java.base/java.lang=ALL-UNNAMED \
     -jar backend.jar \
     --spring.config.additional-location=backend-kafka-config.yml
```
