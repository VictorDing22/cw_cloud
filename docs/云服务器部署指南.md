# 🚀 工业健康监测平台 - 阿里云Ubuntu 20.04部署指南

## 📋 当前环境确认

✅ **云厂商**: 阿里云ECS  
✅ **操作系统**: Ubuntu 20.04 LTS  
✅ **连接工具**: XShell SSH客户端  

### 1. 服务器配置要求

**最低配置**：
- CPU: 2核
- 内存: 4GB
- 硬盘: 40GB SSD
- 带宽: 3Mbps

**推荐配置**（完整功能）：
- CPU: 4核
- 内存: 8GB
- 硬盘: 100GB SSD
- 带宽: 5Mbps

### 2. 需要开放的端口

| 端口 | 服务 | 说明 |
|------|------|------|
| 22 | SSH | 远程连接 |
| 80 | HTTP | Web访问 |
| 443 | HTTPS | 安全访问 |
| 3000 | Frontend | 前端开发服务器 |
| 3306 | MySQL | 数据库 |
| 6379 | Redis | 缓存 |
| 8848 | Nacos | 注册中心 |
| 9092 | Kafka | 消息队列 |
| 48080 | Gateway | API网关 |
| 48081-48084 | 微服务 | 业务服务 |

   端口: 22
   ```
3. **用户身份验证**：
   ```
   方法: Password
   用户名: root (或 ubuntu)
   密码: 你的实例密码

## 📦 第三步：安装数据库和中间件

### 3.1 安装MySQL 8.0

```bash
# 安装MySQL服务器
sudo apt install -y mysql-server

# 启动并设置开机自启
sudo systemctl start mysql
sudo systemctl enable mysql

# 检查MySQL状态
sudo systemctl status mysql

# 安全配置MySQL
sudo mysql_secure_installation
```

**MySQL安全配置步骤**：
1. 密码复杂度验证：n（不需要）
2. 设置root密码：`123456`
3. 移除匿名用户：Y
4. 禁止root远程登录：n（我们需要远程连接）
5. 移除test数据库：Y
6. 重新加载权限表：Y

**配置MySQL数据库**：
```bash
# 登录MySQL（使用刚设置的密码）
mysql -u root -p
# 输入密码：123456
```

**在MySQL命令行中执行以下SQL**：
```sql
-- 创建项目数据库
CREATE DATABASE ruoyi_vue_pro CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE DATABASE nacos_config CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 配置root用户远程连接
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123456';
CREATE USER 'root'@'%' IDENTIFIED WITH mysql_native_password BY '123456';
GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' WITH GRANT OPTION;

-- 刷新权限
FLUSH PRIVILEGES;

-- 查看创建的数据库
SHOW DATABASES;

-- 退出MySQL
EXIT;
```

**测试数据库连接**：
```bash
# 测试连接
mysql -u root -p123456 -e "SHOW DATABASES;"
```

**应该看到输出**：
```
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| nacos_config       |
| performance_schema |
| ruoyi_vue_pro      |
| sys                |
+--------------------+
```

## 🔐 第五步：配置阿里云安全组

### 5.1 在阿里云控制台配置安全组

1. **登录阿里云控制台** → ECS → 实例
2. **点击你的实例** → 安全组 → 配置规则
3. **添加入方向规则**：

| 端口范围 | 授权对象 | 描述 |
|---------|---------|------|
| 22/22 | 0.0.0.0/0 | SSH连接 |
| 80/80 | 0.0.0.0/0 | HTTP访问 |
| 443/443 | 0.0.0.0/0 | HTTPS访问 |
| 3000/3000 | 0.0.0.0/0 | 前端开发服务器 |
| 8848/8848 | 0.0.0.0/0 | Nacos控制台 |
| 48080/48080 | 0.0.0.0/0 | API网关 |
| 3306/3306 | 你的IP/32 | MySQL（仅你的IP） |

### 5.2 配置Ubuntu防火墙

```bash
# 检查防火墙状态
sudo ufw status

# 如果防火墙是激活的，添加规则
sudo ufw allow 22
sudo ufw allow 80
sudo ufw allow 443
sudo ufw allow 3000
sudo ufw allow 8848
sudo ufw allow 48080

# 重新加载防火墙
sudo ufw reload
```

## 🎯 第八步：创建启动脚本

### 8.1 创建服务启动脚本

```bash
cd /opt/cw-cloud

# 创建日志和PID目录
mkdir -p logs pids

# 创建启动脚本
cat > start-services.sh << 'EOF'
#!/bin/bash

echo "=== 启动工业健康监测平台 ==="
echo "时间: $(date)"

# 创建必要目录
mkdir -p logs pids

# 检查基础服务
echo "检查基础服务状态..."
systemctl is-active mysql || { echo "MySQL未启动，请先启动MySQL"; exit 1; }
systemctl is-active redis-server || { echo "Redis未启动，请先启动Redis"; exit 1; }

# 检查Nacos
if ! pgrep -f "nacos" > /dev/null; then
    echo "启动Nacos..."
    cd /opt/nacos/bin
    ./startup.sh -m standalone
    sleep 10
fi

# 等待Nacos就绪
echo "等待Nacos启动完成..."
for i in {1..30}; do
    if curl -s http://localhost:8848/nacos > /dev/null; then
        echo "Nacos已就绪"
        break
    fi
    echo "等待Nacos启动... ($i/30)"
    sleep 2
done

# 启动后端微服务
echo "启动后端微服务..."
cd /opt/cw-cloud/yudao-cloud
nohup java -jar -Xms1g -Xmx2g yudao-server/target/yudao-server.jar --spring.profiles.active=local > ../logs/backend.log 2>&1 &
echo $! > ../pids/backend.pid
echo "后端服务PID: $(cat ../pids/backend.pid)"

# 等待后端启动
echo "等待后端服务启动..."
sleep 30

# 启动前端
echo "启动前端服务..."
cd /opt/cw-cloud/yudao-ui-admin-vue3
nohup npm run dev -- --host 0.0.0.0 > ../logs/frontend.log 2>&1 &
echo $! > ../pids/frontend.pid
echo "前端服务PID: $(cat ../pids/frontend.pid)"

echo ""
echo "=== 启动完成 ==="
echo "前端地址: http://$(curl -s ifconfig.me):3000"
echo "Nacos控制台: http://$(curl -s ifconfig.me):8848/nacos"
echo "API网关: http://$(curl -s ifconfig.me):48080"
echo ""
echo "默认登录账号: admin / admin123"
echo ""
echo "查看日志:"
echo "  后端: tail -f /opt/cw-cloud/logs/backend.log"
echo "  前端: tail -f /opt/cw-cloud/logs/frontend.log"
EOF

chmod +x start-services.sh
```

### 8.2 创建停止脚本

```bash
cat > stop-services.sh << 'EOF'
#!/bin/bash

echo "=== 停止工业健康监测平台 ==="

# 停止前端
if [ -f pids/frontend.pid ]; then
    echo "停止前端服务..."
    kill $(cat pids/frontend.pid) 2>/dev/null
    rm -f pids/frontend.pid
fi

# 停止后端
if [ -f pids/backend.pid ]; then
    echo "停止后端服务..."
    kill $(cat pids/backend.pid) 2>/dev/null
    rm -f pids/backend.pid
fi

# 强制停止相关进程
echo "清理残留进程..."
pkill -f "yudao-server" 2>/dev/null
pkill -f "npm.*dev" 2>/dev/null
pkill -f "node.*vite" 2>/dev/null

# 停止Nacos（可选）
# pkill -f "nacos" 2>/dev/null

echo "所有服务已停止"
EOF

chmod +x stop-services.sh
```

### 8.3 创建状态检查脚本

```bash
cat > check-status.sh << 'EOF'
#!/bin/bash

echo "=== 系统状态检查 ==="
echo "时间: $(date)"
echo ""

echo "=== 基础服务状态 ==="
echo "MySQL: $(systemctl is-active mysql)"
echo "Redis: $(systemctl is-active redis-server)"
echo "Nacos: $(pgrep -f nacos > /dev/null && echo "运行中" || echo "未运行")"
echo ""

echo "=== 应用服务状态 ==="
if [ -f pids/backend.pid ] && kill -0 $(cat pids/backend.pid) 2>/dev/null; then
    echo "后端服务: 运行中 (PID: $(cat pids/backend.pid))"
else
    echo "后端服务: 未运行"
fi

if [ -f pids/frontend.pid ] && kill -0 $(cat pids/frontend.pid) 2>/dev/null; then
    echo "前端服务: 运行中 (PID: $(cat pids/frontend.pid))"
else
    echo "前端服务: 未运行"
fi
echo ""

echo "=== 端口监听状态 ==="
netstat -tlnp 2>/dev/null | grep -E "(3000|3306|6379|8848|48080)" | while read line; do
    port=$(echo $line | awk '{print $4}' | cut -d: -f2)
    echo "端口 $port: 监听中"
done
echo ""

echo "=== 内存使用情况 ==="
free -h
echo ""

echo "=== 磁盘使用情况 ==="
df -h /
EOF

chmod +x check-status.sh
```

### 完整部署命令（复制粘贴执行）

```bash
# 1. 更新系统并安装基础软件

# 2. 配置环境变量
echo 'export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc

# 3. 安装Maven
cd /tmp && wget https://downloads.apache.org/maven/maven-3/3.9.5/binaries/apache-maven-3.9.5-bin.tar.gz
sudo tar -xzf apache-maven-3.9.5-bin.tar.gz -C /opt/
sudo ln -s /opt/apache-maven-3.9.5 /opt/maven
echo 'export MAVEN_HOME=/opt/maven' >> ~/.bashrc
echo 'export PATH=$PATH:$MAVEN_HOME/bin' >> ~/.bashrc
source ~/.bashrc

# 4. 配置MySQL（需要先运行 mysql_secure_installation 设置密码为 123456）
mysql -u root -p123456 -e "CREATE DATABASE ruoyi_vue_pro CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
mysql -u root -p123456 -e "CREATE DATABASE nacos_config CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
mysql -u root -p123456 -e "ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123456';"
mysql -u root -p123456 -e "CREATE USER 'root'@'%' IDENTIFIED WITH mysql_native_password BY '123456';"
mysql -u root -p123456 -e "GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' WITH GRANT OPTION;"
mysql -u root -p123456 -e "FLUSH PRIVILEGES;"

# 5. 安装Nacos
cd /opt && sudo wget https://github.com/alibaba/nacos/releases/download/2.2.3/nacos-server-2.2.3.tar.gz
sudo tar -xzf nacos-server-2.2.3.tar.gz && sudo chown -R $USER:$USER nacos

# 6. 创建项目目录
sudo mkdir -p /opt/cw-cloud && sudo chown $USER:$USER /opt/cw-cloud
```
