## 🎯 安装步骤（共5步）

### 步骤1: 下载 Redis ⭐

1. **打开浏览器**，访问:
   ```
   https://github.com/tporadowski/redis/releases
   ```

2. **下载文件**: `Redis-x64-5.0.14.1.zip` (约3MB)

3. **解压**: 解压到 `C:\Redis`
   - 确保路径为: `C:\Redis\redis-server.exe`

---

### 步骤2: 下载 Nacos ⭐

1. **打开浏览器**，访问:
   ```
   https://github.com/alibaba/nacos/releases
   ```

2. **下载文件**: `nacos-server-2.2.3.zip` (约100MB)

3. **解压**: 解压到 `C:\Nacos`
   - 确保路径为: `C:\Nacos\nacos\bin\startup.cmd`

---

### 步骤3: 安装服务

**以管理员身份**运行: `安装Redis和Nacos.bat`

- 这个脚本会检测安装并注册Redis服务
- 按照提示确认安装

---

### 步骤4: 初始化数据库

运行: `初始化数据库.bat`

- 输入MySQL密码
- 自动创建数据库和表结构
- 创建: `ruoyi_vue_pro` 和 `nacos_config` 数据库

---

### 步骤5: 配置并启动 Nacos

**5.1 配置Nacos**

运行: `配置Nacos.bat`

- MySQL地址: localhost
- MySQL端口: 3306
- MySQL用户: root
- MySQL密码: (输入您的密码)
- 选择 Y 初始化Nacos数据库表

**5.2 启动Nacos**

运行: `2-启动Nacos.bat`

- 等待约30秒
- 访问: http://localhost:8848/nacos
- 用户名: nacos
- 密码: nacos

## 🐛 常见问题

### 1. Redis服务启动失败?
- 以管理员身份运行脚本
- 检查端口6379是否被占用

### 2. Nacos启动失败?
- 确认MySQL中nacos_config数据库已创建
- 检查配置文件中的数据库密码
- 查看日志: `C:\Nacos\nacos\logs\start.out`

### 3. 前端无法连接后端?
- 确认Gateway已启动 (端口48080)
- 检查Nacos中是否注册了服务
- 查看浏览器控制台的错误信息
