## 🔍 故障排查

### 问题A: TDMS生产者无数据

**检查**:
```batch
cd kafka
bin\windows\kafka-console-consumer.bat --bootstrap-server localhost:9092 --topic sample-input --timeout-ms 3000
```

**如果没有数据**:
1. 检查TDMS生产者窗口是否有错误
2. 确认 `floatdata\data\*.tdms` 文件存在
3. 重启TDMS生产者: `START-TDMS-REALTIME.bat`

---

### 问题B: 前端显示模拟数据

**症状**: 设备ID是 `sensor-simulation`

**原因**: Backend没有向Kafka发送数据

**解决**:
1. 检查sample-output topic是否有数据
2. 重启Backend: `START-BACKEND-FAST.bat`
3. 查看Backend控制台是否有错误

---

### 问题C: Kafka启动失败

**症状**: `all log dirs have failed`

**解决**:
```batch
# 1. 停止所有Kafka进程
taskkill /F /IM java.exe /FI "MEMUSAGE lt 600000"

# 2. 删除数据目录
rmdir /S /Q data\kafka-logs
rmdir /S /Q data\zookeeper

# 3. 重新启动
KAFKA-START.bat
```
